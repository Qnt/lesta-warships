# Леста Корабли

Тестовое задание от "Леста"

## Установка и запуск

1. Склонировать репозиторий.

```bash
git clone https://github.com/Qnt/lesta-warships.git
```

2. Установить зависимости.

```bash
npm install
```

3. Определить переменные в .env (см. .env.example).
4. Запустить.

```bash
npm run start
```
